!!! gird-template-columns/rows -> 명시적인 선언이다! 만약 명시적선언의 개수의 이외 것들을 암시적선언은 밑에 속성사용! 1. gird-auto-columns : -> 명시적 선언없이 사용가능 예1) grid-template-columns : 100px 100px; grid-auto-columns : 200px 위의 코드는 가로에 두개에대한 100px 아이템크기만 정의되있다. 즉 template 로 명시적 선언한것만이다! 그렇다면 다음 아이템들의 크기는 content 크기에 맞춰서 각각 다르다. 그외 것들은 auto로 암시적 선언을사용하여 200px 유지할수있다. 내용이추가된다고해도 template 에 더추가할수고를 덜어줌 !! 가로크기에 몇개가 오는것을 모른다고 명시적선언없이 사용할경우 각각의 아이템들에 grid-column : 2/3 등등 각아이템들에 값을줘야하는 번거로움이발생함 grid-auto-columns: repeat(auto-fill,minmax(25%, auto)); 위와같이 명시적인 값또한 사용불가함 2. grid-auto-rows : -> 명시적 선언없이 사용가능 예1 ) grid-template-rows : repeat(3, minmax(100px, auto)) 세로에 대한 정의를 최소는 100px 최대는 auto다 그런데 세로 셀이늘어난다면?
